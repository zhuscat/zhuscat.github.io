继承属性与非继承属性

发表于 2016-12-12 | 分类于 学习笔记 |

CSS 属性可以分为继承属性与非继承属性。这决定着当某一属性没有提供值得时候，该属性的值为什么。

继承属性

CSS 继承属性是在没有给元素指定某一属性的时候，该属性的值为属性父元素该值得计算值。
